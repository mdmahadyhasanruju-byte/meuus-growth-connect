## 11. Accessibility and Usability Notes

Verified or implemented improvements include:

- a skip link targeting `#main-content`;
- semantic main-content targeting;
- popup dialog labelling and modal semantics;
- initial popup focus and focus restoration on dismissal;
- Escape-key dismissal;
- visible keyboard focus styles for popup controls;
- a descriptive accessible label and visible focus style for the floating Book Preview CTA;
- the floating CTA remaining hidden on `/book` and `/books/*`; and
- removal of the `/global` horizontal overflow caused by a decorative orb.

The browser-control surface did not reliably reproduce synthetic Tab movement during live verification. The skip-link target and focus implementation were verified in the deployed DOM and code, but this report is not a substitute for a full manual keyboard audit.

## 12. Performance and Asset Notes

- No public asset above 20 MiB was found during the audit.
- The current v0.3 book PDF is approximately 5.95 MiB.
- The particle canvas caps high-DPI rendering at a maximum 2× pixel ratio.
- Continuous particle animation pauses while the document is hidden.
- Reduced-motion users receive a static decorative frame.
- Normal particle rendering remained active after deployment.
- Existing server and runtime error logging was retained as intentional diagnostics.

## 13. Known Limitations

- The Book Preview popup does not reappear after dismissal unless its localStorage key is cleared.
- Reduced-motion behavior was implemented and build-validated, but live browser preference emulation was limited.
- Synthetic Tab movement was not fully reliable in the browser-control environment.
- This report is not a formal accessibility certification.
- This report is not a penetration test or full security audit.
- This report is not a Phase One launch report.
- It does not validate DLAS, AI, professional services, or the wider ecosystem as operational.
