# One Word Start Manual Pilot

## Non-Negotiable Boundary

One Word Start is a proposed manual pilot. It is:

- Not AI.
- Not DLAS.
- Not diagnosis.
- Not therapy.
- Not automated guidance.
- Not a mental-health assessment.
- Not evidence of identity, ability, risk, readiness, or personal truth.

## Minimum Governance Required Before Use

One Word Start must not accept participant input until all of the following exist:

- Consent rule.
- Named owner.
- Retention rule.
- Deletion rule.
- Stop condition.
- Evidence form.
- Participant scope.
- Prohibited-data list.
- Permission-before-guidance language.
- Escalation and emergency redirection boundary.

## Manual Pilot Shape

1. Invite a minimal signal.
2. State that the signal can be incomplete.
3. Ask whether the person wants to add context.
4. Ask permission before any guidance.
5. Offer only approved manual next steps.
6. Record process evidence and uncertainty.
7. Stop, delete, or retain only according to the approved rules.

## Success Definition

Early success is not scale, transformation, clinical impact, personalization, or AI capability. Early success is limited to proof that consent was captured, boundaries were honored, participants could stop, prohibited data was avoided or deleted, and claims matched records.
